Searches file contents using patterns and regex.
Parameters:
- pattern: Search pattern (regex supported, optional)
- directory: Directory to search (defaults to current directory)
- include: File patterns to include (e.g., "*.py", "*.js")

Examples: 
- {"pattern": "TODO"} - find files containing "TODO"
- {"pattern": "password|secret|key", "include": "*.py"} - find secrets in Python files
- {"pattern": "function.*login", "directory": "src"} - find login functions in src directory

Returns paths of files containing matches, sorted by modification time.
